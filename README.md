# AI YouTube TV Roku Assistant

This repository will contain a private AI TV assistant for a Roku-branded TV. The project is being built in deliberate phases rather than as a one-shot generated application.

The current phase is **architecture only**. See [`docs/architecture.md`](docs/architecture.md) for the system design, phase plan, subsystem boundaries, and open implementation decisions.

## Current phase

- Architecture documented.
- No runtime backend, database schema, Roku channel, planner, updater, or tests have been generated yet.
- The next phase should create the folder structure and tooling skeleton only.
