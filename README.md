# AI YouTube TV Roku Assistant

This repository will contain a private AI TV assistant for a Roku-branded TV. The project is being built in deliberate phases rather than as a one-shot generated application.

The architecture phase is complete. The repository now contains the Phase 2 folder structure and development skeleton for future implementation.

## Current phase

- Architecture documented in [`docs/architecture.md`](docs/architecture.md).
- Monorepo directories, placeholder package documentation, configuration examples, tooling configuration, Docker placeholders, CI placeholders, and subsystem documentation have been added.
- No runtime backend, database schema, Roku channel, planner, updater, AI logic, Roku communication, or business logic has been implemented.
- The next phase should implement the database only.
